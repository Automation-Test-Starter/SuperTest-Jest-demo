# SuperTest-Jest-demo

一个使用 Jest 的 SuperTest API 自动化测试演示项目

## 环境准备
- Nodejs ，我使用的 v21.1.0
- SuperTest 我使用的 6.3.3 版本
- Jest 我使用的是最新的 29.7.0 版本

## 技术栈
- SuperTest
- Jest
- jest-html-reporters
- GitHub action

## 项目运行

```bash
node run test
```

## 添加API测试用例

- 在 Specs目录下新加测试用例即可
